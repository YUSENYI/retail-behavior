## 负责的项目模块

### 后端服务模块（backend/）

1. **权限与审计服务** (backend/src/api/routes/)
   - 识别当前操作者
   - 校验角色权限
   - 记录关键操作日志
   - 对敏感字段进行脱敏

2. **预警服务** (backend/src/analytics/alert_service.py)
   - 检测指标异常
   - 检测转化异常
   - 检测低活跃和流失风险
   - 记录预警处理状态

3. **报表服务** (backend/src/api/routes/report.py)
   - 创建报表任务
   - 生成报表快照
   - 记录导出状态
   - 管理导出文件路径

### 前端应用模块（frontend/）

1. **审计日志模块** (frontend/src/pages/admin/audit/)
   - 角色访问日志查看
   - 查询日志查看
   - 导出日志查看
   - 系统配置修改日志查看
   - 敏感数据访问日志查看

2. **预警中心模块** (frontend/src/pages/admin/alert/)
   - 低活跃提醒展示
   - 流失风险预警展示
   - 转化异常预警展示
   - 行为数据异常预警展示
   - 预警处理状态流转

3. **报表统计模块** (frontend/src/pages/admin/report/)
   - 用户行为明细报表
   - 商品分析报表
   - 转化漏斗报表
   - 用户画像报表
   - 用户分群报表
   - 销售转化报表
   - 运营效果报表

### 测试模块

1. **后端接口测试** (backend/tests/)
   - API 接口测试用例
   - 业务逻辑测试
   - 权限测试
   - 数据准确性测试

2. **前端端到端测试** (frontend/tests/e2e/)
   - 用户购物端流程测试
   - 运营管理端流程测试
   - 权限控制测试
   - 预警处理流程测试

3. **视觉回归测试** (frontend/tests/visual/)
   - 图表渲染正确性测试
   - 页面布局一致性测试

## 测试方案

### 后端接口测试（pytest）

- **测试框架**：pytest
- **测试范围**：
  - API 接口正确性
  - 业务逻辑正确性
  - 权限控制正确性
  - 数据准确性（非负、比例在0-100之间）

### 前端端到端测试（Playwright）

- **测试框架**：Playwright
- **测试范围**：
  - 用户购物端完整流程（浏览→加购→下单→支付）
  - 运营管理端完整流程（登录→查看数据→处理预警→导出报表）
  - 权限控制流程（不同角色看到不同菜单和数据）
  - 预警处理流程（触发预警→查看预警→处理预警）

### 视觉回归测试（Playwright）

- **测试框架**：Playwright + 截图对比
- **测试范围**：
  - 图表渲染正确性（漏斗图、饼图、柱状图、趋势图）
  - 页面布局一致性（不同分辨率下的布局）

## 质量标准

### 代码质量

- 单元测试覆盖率 > 80%
- API 接口测试覆盖率 > 90%
- 端到端测试覆盖核心业务流程

### 性能标准

- API 接口响应时间 < 500ms（95分位）
- 页面加载时间 < 2s（首次加载）
- 图表渲染时间 < 1s

### 安全标准

- 所有敏感数据默认脱敏展示
- 所有关键操作记录操作日志
- 所有接口进行权限校验
